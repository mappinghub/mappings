* [CareTeam](https://www.hl7.org/fhir/careteam.html)
* [Goal](https://www.hl7.org/fhir/goal.html)
* [ReferralRequest](https://www.hl7.org/fhir/referralrequest.html)
* [ProcedureRequest](https://www.hl7.org/fhir/procedurerequest.html)
* [NutritionOrder](https://www.hl7.org/fhir/nutritionorder.html)
* [VisionPrescription](https://www.hl7.org/fhir/visionprescription.html)
* [RiskAssessment](https://www.hl7.org/fhir/riskassessment.html)
* [RequestGroup](https://www.hl7.org/fhir/requestgroup.html)

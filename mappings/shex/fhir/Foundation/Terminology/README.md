# Terminology 

This directory holds FHIR Terminology mappings for translating from common ShEx to FHIR:
* [CodeSystem](https://www.hl7.org/fhir/codesystem.html)
* [ConceptMap](https://www.hl7.org/fhir/conceptmap.html)
* [ExpansionProfile](https://www.hl7.org/fhir/expansionprofile.html)
* [NamingSystem](https://www.hl7.org/fhir/namingsystem.html)
* [ValueSet](https://www.hl7.org/fhir/valueset.html)

This directory holds FHIR Entities mappings for translating from common ShEx to FHIR:

* [HealthcareService](https://www.hl7.org/fhir/healthcareservice.html)
* [Endpoint](https://www.hl7.org/fhir/endpoint.html)
* [Location](https://www.hl7.org/fhir/location.html)
* [Substance](https://www.hl7.org/fhir/substance.html)
* [Device](https://www.hl7.org/fhir/device.html)
* [DeviceComponent](https://www.hl7.org/fhir/devicecomponent.html)
* [DeviceMetric](https://www.hl7.org/fhir/devicemetric.html)
